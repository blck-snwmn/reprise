# Reprise

A Chrome extension for looping specific sections of YouTube videos.

## Features

- Loop playback with custom start/end times
- Configure via popup UI
- Toggle loop ON/OFF

## Build

```bash
bun install
bun run build
```

## Installation

1. Download zip from [Releases](https://github.com/blck-snwmn/reprise/releases) or run `bun run build`
2. Open `chrome://extensions`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the extracted folder or `dist/`
